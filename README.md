## 安装

1. 游戏内打开 **卫月设置** → **插件** → **第三方插件仓库**
2. 添加以下链接并启用：

```
https://gitee.com/baofengz/tech-guide-repo/raw/main/pluginmaster.json
```

3. 打开 **插件安装器**，搜索 **TechGuide** 并安装

## 仓库内容

| 文件 | 说明 |
|------|------|
| `pluginmaster.json` | 插件索引 |
| `release/TechGuide/latest.zip` | 安装包 |
